# 🎨 ThemeStore - Warm Nordic Premium CSS Templates Store

A premium CSS and UI template store web application designed with a unique, human-crafted **Warm Nordic (Sand, Terracotta & Sage)** light theme, deviating from generic templates.

## ✨ Humanized & Localized Features
- 🇩🇰 **Warm Nordic Theme** — Styled using Lora (Serif) & Outfit (Sans-Serif) typography on a warm sand background with terracotta rust and sage green accents.
- 💳 **Lipa na M-Pesa Checkout Flow** — Click "Buy Now" on any premium template to trigger a simulated STK Push checkout dialog, including phone validation and M-Pesa PIN simulation.
- 👨‍💻 **Custom Integration Banner** — Prominent top-bar notification helping users contact Fred Otieno (+254782091381) directly for tailor-making templates.
- 🎬 **Cinematic Header Background** — Dynamic CSS zoom-slideshow rendering technical workspaces in the background.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Libraries:** Font Awesome
- **Typography:** Lora (Headings), Outfit (Body/Labels)
